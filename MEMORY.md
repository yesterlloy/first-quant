# 量化系统项目记录

## 项目：A股因子选股量化分析系统

### 关键决策
- 目标市场：A股为主，后期扩展其他市场
- 策略风格：因子选股
- 数据源：优先免费（akshare/tushare），质量不够再付费
- 部署：Docker，本机服务器
- 开发节奏：先跑通原型再迭代

### 阶段规划
- Phase 1：数据采集 + 存储 + 基础回测原型 ✅ 已完成 (2026-05-29)
- Phase 2：因子系统 + 因子检验 ✅ 已完成 (2026-05-29)
- Phase 3：ML增强 + 因子合成
- Phase 4：实盘对接 + 风控监控

- akshare/baostock/adata多数据源采集器，增量更新+失败重试+自动降级

- Docker部署一键启动
- 核心代码约1485行
